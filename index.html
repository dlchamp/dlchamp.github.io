Save time with the CSV,Roles!

Invite to Server

Prequisites...

  • Bot's top role MUST be higher than any role that you plan to bulk manage
  • The CSV file MUST be named "members.csv" - This prevents the bot from accidentally trying to parse other CSV uploads.
  • Required Permissions:
    • Manage Roles - adding roles to members
    • View Channel/Read Messages - seeing and parsing your messages and attachments
    • Send Message/Manage Messages - sending progress and completion messages/delete original message and attachment
    • Add Reactions - adding reaction to final message to allow command user to delete completion/error messages

How it works...

  • Migrate your data to a CSV file
  • Make sure your member name/ids are the second column
  • Head into a Discord channel the bot can see and read messages in
  • Copy the role ID for the role you wish to assign your members
  • Paste it into the message box and attach your members.csv file
  • Send your message and wait for CSVRoles to work it's magic.
